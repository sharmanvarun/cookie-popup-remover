# Cookie Popup Remover — User Guide

This guide is for people who have never used GitHub or worked with code before.
You do **not** need to understand programming to use this tool.

## What this tool does

It removes cookie pop-ups and lets you scroll and click the page again.
It is most useful on sites that block the whole page until you accept cookies.

## What you need

- A web browser such as Chrome, Edge, or Firefox
- Internet access
- About 5 minutes

## Step 1: Download the files from GitHub

1. Open the GitHub page for this project in your browser.
2. Find the green **Code** button.
3. Click **Code**.
4. Click **Download ZIP**.
5. Save the ZIP file somewhere easy to find, like **Downloads**.

## Step 2: Unzip the download

1. Open your **Downloads** folder.
2. Find the ZIP file you just downloaded.
3. Right-click it and choose **Extract All** or **Unzip**.
4. Open the folder that was created.

## Step 3: Open the install page

1. Open the folder you unzipped.
2. Open the **bookmarklet** folder.
3. Double-click **bookmarklet.html**.

This opens a simple page with a brown button.

## Step 4: Put it in your bookmarks

### Easiest way

1. Make sure the bookmarks bar is visible:
   - Chrome / Edge / Firefox: press **Ctrl + Shift + B**
2. Drag the brown **Remove cookie pop-up** button to the bookmarks bar.

### If dragging does not work

1. Right-click the bookmarks bar.
2. Choose **Add page** or **New bookmark**.
3. Name it **Remove cookie pop-up**.
4. Copy the long address from the page and paste it into the URL field.

## Step 5: Use it

1. Go to a website that shows a cookie wall.
2. Click the **Remove cookie pop-up** bookmark.
3. The pop-up should disappear and the page should scroll normally again.

## If the pop-up comes back

Some websites show the cookie message again later. If that happens, just click the bookmark again.

## Optional: browser extension

If you want the tool to run automatically, you can use the browser extension version.
This is more advanced than the bookmarklet.

### Chrome / Edge

1. Open `chrome://extensions` or `edge://extensions`.
2. Turn on **Developer mode**.
3. Click **Load unpacked**.
4. Choose the **extension** folder from this project.

### Firefox

1. Open `about:debugging#/runtime/this-firefox`.
2. Click **Load Temporary Add-on**.
3. Open the **extension** folder from this project.
4. Choose the **manifest.json** file.

## If something goes wrong

- If the page does not change, refresh the page and click the bookmark again.
- If the bookmark is missing, show the bookmarks bar again with **Ctrl + Shift + B**.
- If you are on a work or school computer, the extension may be blocked. Use the bookmarklet instead.
